# Communes Project

## What this is
A Meetup-like platform where people find like-minded communities in their cities.

## Tech stack
- Next.js 14 (App Router)
- Supabase (database + auth)
- Tailwind CSS

## Core features
- Browse communes by category and city
- Create a commune
- Join a commune
- View commune members

## Code standards
- Use TypeScript strictly, no `any` types
- Use server components by default, client components only when needed
- Keep components small and focused
- Use Tailwind for all styling, no custom CSS files
